API Reference - Server Scoped
    ServiceNow provides JavaScript APIs for use within scripts running on the ServiceNow platform to deliver common functionality. 
This reference lists available classes and methods along with parameters, descriptions, and examples to make extending the 
ServiceNow platform easier.

    ServiceNow 提供 JavaScript API 以在 ServiceNow 平台上运行的脚本中使用，以提供通用功能。 此参考列出了可用的类和方法以及参数、描述和示例，
以便更轻松地扩展 ServiceNow 平台。
    ServiceNowは、ServiceNowプラットフォームで実行されているスクリプト内で使用して共通の機能を提供するためのJavaScriptAPIを提供します。 
このリファレンスには、ServiceNowプラットフォームの拡張を容易にするためのパラメーター、説明、および例とともに、使用可能なクラスとメソッドが
リストされています。
